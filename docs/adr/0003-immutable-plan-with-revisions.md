# Immutable Plan with dated Revisions

A Position begins life as a Plan (thesis, intended structure, max risk, targets) before any Execution exists. The original Plan is never editable; deliberate changes of intent are recorded as dated Plan Revisions. This keeps "original risk/reward" honest — it can never be quietly rewritten after the market moves — while still allowing legitimate re-planning (e.g., after legging into a spread). An editable plan was rejected because it destroys the behavioral-discipline value of plan-vs-outcome comparison.
